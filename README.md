# 🌌 XENO_OS_MASTER_Ω

XENO_OS_MASTER_Ω is a high-performance, mobile-optimized HUD (Heads-Up Display) interface inspired by the **Flipper Zero** and **Alien Technology** aesthetics. Designed specifically for developers and power users who operate without a desktop, providing a futuristic terminal experience right in the browser.

## 🚀 Features

* **Cyber-Alien UI:** A stunning interface with plasma purple and alien cyan color schemes.
* **Mobile First:** Fully responsive design with touch-optimized controls and haptic feedback (vibration).
* **Flipper Style Controls:** Includes a virtual D-pad and a "Master Deploy" execution button.
* **Simulated Modules:** * `NMAP` - Network Scanning Interface
    * `SHELL` - Root Terminal Access
    * `WIFI` - Handshake Interceptor
    * `AIM` - Gaming Assistance HUD (Free Fire Optimized)
* **Interactive Core:** Rotating neural rings with real-time particle generation.

## 🛠️ Built With

* **HTML5** - Semantic structure.
* **CSS3** - Custom animations, glassmorphism, and neon effects.
* **JavaScript (Vanilla)** - Dynamic node generation and system simulation.

## 📱 How to Use on Mobile

1.  **Clone/Download:** Upload the `index.html` to your GitHub repository.
2.  **Deployment:** Enable **GitHub Pages** in your repository settings.
3.  **Access:** Open the generated link on your mobile browser (Chrome/Safari).
4.  **Interaction:** Tap the alien symbols to trigger simulated modules or use the D-pad for navigation.

## ⚠️ Disclaimer

This project is a **UI/UX Simulation**. It is intended for educational purposes and as a front-end template for security enthusiasts. It does not perform actual "hacking" activities but provides a base for integrating real APIs via Termux or Node.js.
